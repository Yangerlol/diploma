Дипломный проект: Автоматизация тестирования сервиса "AQA Shop"
Описание проекта
Данный проект представляет собой автоматизацию тестирования комплексного сервиса покупки туров. Сервис взаимодействует с СУБД и банковскими API.

Технологический стек
Язык программирования: Java 21.

Тестовая среда: JUnit 5.

Фреймворк автоматизации: Selenide (Selenium).

Сборка проекта: Gradle.

Базы данных: PostgreSQL / MySQL (запуск через Docker Desktop).

Отчетность: Allure.

Окружение для тестирования
ОС: Windows 10.

Браузер: Google Chrome v143.0.7499.170.

Как запустить тесты
Склонируйте репозиторий.

Запустите контейнеры с базами данных и эмулятором банковских сервисов через Docker.

В терминале IntelliJ IDEA выполните команду для запуска тестов: .\gradlew.bat clean test.

Результаты тестирования
В ходе выполнения проекта были автоматизированы сценарии оплаты по карте (PaymentTest.java):

Успешная оплата: Тест пройден, данные карты корректно отправлены в банк.

Отказ при оплате: Тест пройден, система корректно обрабатывает отклоненные операции.
